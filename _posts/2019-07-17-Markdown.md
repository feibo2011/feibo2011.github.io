# Markdown

md的使用：

```
1.标题层级如下，最多6级
# 顶级标题 等价于 title 和 <h1>
## 次级标题 等价于 <h2>
### 3级标题 等价于 <h3>
#### 4级标题 等价于 <h4>
##### 5级标题 等价于 <h5>
###### 6级标题 等价于 <h6>
####### 注：此行错误，标题层级最多6级 没有<h7>标签


2.加强和强调规范
 *强调 **加强  							*emphasize*   **strong** 
使用~~给文字添加删除线				~~strikethrough~~
3个波浪						块级代码
 链接    An [example](http://url.com/ "Title")
```
